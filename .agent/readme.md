## Task

Make the **Search** button and search bar fully functional.

Requirements:

1. Locate the current search form.
2. Ensure the form submits correctly.
3. Connect it to the appropriate controller.
4. Search real data from the database (do not use mock data).
5. Return matching results.
6. If no result exists, display a friendly "No results found" message.
7. Preserve the current UI as much as possible.
8. Prevent SQL injection by using the project's existing query/prepared statement pattern.
